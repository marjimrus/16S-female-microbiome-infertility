# Data Access

## Raw Sequencing Data

Raw 16S rRNA sequences are publicly available at NCBI SRA:

- **BioProject**: [PRJNA1031834](https://www.ncbi.nlm.nih.gov/bioproject/PRJNA1031834)
- **SRA Study**: SRP123456

### Download sequences from NCBI

```bash
# Install SRA Toolkit if needed
# conda install -c bioconda sra-tools

# Download all samples from the BioProject
prefetch PRJNA1031834

# Convert to FASTQ format
fasterq-dump --split-files SRR*
```

### Create manifest file for QIIME2

After downloading, create a manifest file with the following format:

```
sample-id	absolute-filepath	direction
Subject1.EN	/path/to/fastq/SRR_sample1.fastq	forward
Subject1.EF	/path/to/fastq/SRR_sample2.fastq	forward
...
```

Use the `metadata.xlsx` file to match Sample_IDs with SRA accession numbers.

## Metadata

The `metadata.xlsx` file contains anonymized clinical and sample information for all 105 samples (21 patients Ã— 5 body sites).

### Columns description

| Column | Description |
|--------|-------------|
| Sample_ID | Unique sample identifier (SubjectID.SampleType) |
| SubjectID | Anonymized patient identifier |
| Sample_Type_Code | EN=Endometrium, EF=Endometrial fluid, VA=Vagina, OR=Oral, FE=Feces |
| Sample_Type | Full name of sample type |
| Body_Site | FRT (Female Reproductive Tract) or GIT (Gastrointestinal Tract) |
| Gynecological_Diagnosis | Clinical diagnosis |
| Study_Group | Endometriosis or Other_Conditions |
| Age | Patient age at sampling |
| Eggs_Receptor | Whether patient required egg donation (Yes/No) |
| Autoimmune_Disease | Presence of autoimmune disease (Yes/No) |
| Other_Drugs | Use of other medications (Yes/No) |
| Contraceptives | Use of contraceptives (Yes/No) |
| Eutirox | Use of Eutirox/levothyroxine (Yes/No) |
| Alcohol | Alcohol consumption (Yes/No) |
| Tobacco | Tobacco use (Yes/No) |
